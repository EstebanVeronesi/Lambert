3. Fórmulas de Cálculo

3.1 Fase 0 — Preparación de variables

// PBT efectivo (no puede superar el máximo normativo)
pbt = min(pbtEntrada, pbtMaxNormativo)

// Límites máximos por eje
maxEjeDelantero = pbtMaxNormativo × (PORCENTAJE / 100)
maxEjeTrasero = pbtMaxNormativo - maxEjeDelantero

// Suma de todas las cargas extra
cargaExtraTotal = Σ(cargas_extra[i].peso)

// Centro de masa ponderado de cargas extra
momentoTotalCargas = Σ(cargas_extra[i].peso × cargas_extra[i].distancia_eje_delantero)
posCargaExtra = momentoTotalCargas / cargaExtraTotal   (si cargaExtraTotal > 0)

3.2 Fase 1 — Diagnóstico: Fórmula Maestra (Celda N64 de Gian)

// Capacidades útiles de cada eje
capLibreDelantera = maxEjeDelantero - taraDelantera
capLibreTrasera = maxEjeTrasero - taraTrasera
cargaUtilCarroceria = (capLibreDelantera + capLibreTrasera) - cargaExtraTotal

// Centro de carga de la carrocería (desde eje delantero)
centroCargaCaja = espaldaCabina + sepCabina + (largoCarroceria / 2)

// ═══════════════════════════════════════════════════
// FÓRMULA MAESTRA — Equilibrio de momentos sobre eje trasero
// ═══════════════════════════════════════════════════
momentoCarroceria = cargaUtilCarroceria × (distEjes - centroCargaCaja)
momentoTaraDel = taraDelantera × distEjes
momentoCargaExtra = cargaExtraTotal × (distEjes - posCargaExtra)

pesoRealDelantero = (momentoCarroceria + momentoTaraDel + momentoCargaExtra) / distEjes

// Porcentajes reales de distribución
porcentajeRealDelantero = (pesoRealDelantero / pbt) × 100
porcentajeRealTrasero = 100 - porcentajeRealDelantero

// Cargas netas por eje (peso real menos tara)
cargaEjeDelantero = pesoRealDelantero - taraDelantera
cargaEjeTrasero = (pbt - pesoRealDelantero) - taraTrasera

3.3 Fase 2 — Solución: Desplazamiento del eje (Celda N54)

// Momentos totales de las cargas
momentosCargas = (cargaExtraTotal × posCargaExtra) + (cargaUtilCarroceria × centroCargaCaja)
pesoObjetivoTraseroLibre = maxEjeTrasero - taraTrasera

nuevaDistEjes = momentosCargas / pesoObjetivoTraseroLibre   (si > 0, sino distEjes)
desplazamientoEje = nuevaDistEjes - distEjes

3.4 Fase 3 — Geometría: Chasis y voladizo

// Alargar o cortar chasis
espacioNecesario = espaldaCabina + sepCabina + largoCarroceria
espacioDisponible = distEjes + volTrasero
modifChasis = espacioNecesario - espacioDisponible

// Voladizo trasero resultante (sin mover el eje)
voladizoTraseroCalculado = espacioNecesario - distEjes

// Voladizo con desplazamiento del eje
voladizoConDesplazamiento = volTrasero - desplazamientoEje + modifChasis
pctVoladizoConDesplazamiento = (voladizoConDesplazamiento × 100) / nuevaDistEjes

// Largo final del camión armado
largoTotalCamion = volDelantero + distEjes + volTrasero + modifChasis

// Centro de carga total
centroCargaTotal = centroCargaCaja

5.3 Lógica de generación por prioridad

PRIORIDAD 1 — No requiere cambiar el camión
  • Si voladizo excede → reducir largo de carrocería en el exceso exacto calculado
  • Si voladizo excede y hay margen → reducir separación cabina-carrocería
  • Si hay cargas extra → sugerir reposicionarlas para optimizar centro de carga
PRIORIDAD 2 — Requiere modificar el chasis
  • Si voladizo excede y chasis necesita cambio → alargar o cortar chasis
  • Si desplazamiento ≥ 10mm → desplazar eje trasero (adelante o atrás), indicando si el voladizo resultante cumple o no
PRIORIDAD 3 — Requiere cambiar el camión
  • Si porcentaje < mínimo → calcular PBT necesario; si supera máximo normativo → cambiar camión
  • Si porcentaje > máximo → sugerir cambiar a 6x2 o reducir PBT
  • Si carga negativa en algún eje → aumentar PBT o cambiar camión con diferentes taras
